All the courses shown below are available via the MaprunF App that is available for both Android and iOS phones.

A brief introduction to creating your own MaprunF courses can be found [here](/documents/virtual-maps/MapRunF.docx).

    Pendle Forest Orienteers Maprun courses can be found, via the App, in the UK/Lancashire directory.

They are categorised by Geographic locations to help you find the courses that are most convenient for you. A few may be categorised by 'competition' such as the Rossendale Schools Sports Partnership events that are in the UK/Lancashire/Rossendale/RSSP directory.

Any courses that include 'Start Anywhere' in the course name mean that you can start this course at any control on the course rather than everyone starting at the same location.

Brief details of the courses and links to download Paper maps for these events can be found in the table below.
A brief overview of how to use MapRunF can be found [here](/documents/Maprun20updated.docx).
